# Resume Builder 📄

A full-stack Resume Builder application developed using Java, Spring Boot, React, and MySQL. The platform enables users to create, customize, preview, and download professional ATS-friendly resumes through an intuitive and responsive interface.

---

## 🚀 Features

### User Features

* User Registration & Login
* Create Multiple Resumes
* Edit and Update Resume Details
* Professional Resume Templates
* Real-Time Resume Preview
* PDF Resume Download
* Responsive User Interface
* Resume Management Dashboard

### Resume Sections

* Personal Information
* Education Details
* Work Experience
* Projects
* Technical Skills
* Certifications
* Achievements
* Social Links (LinkedIn, GitHub)

### Future Enhancements

* JWT Authentication & Authorization
* AI-Powered Resume Suggestions
* Cover Letter Generator
* Premium Resume Templates
* Razorpay Payment Integration
* Cloud Storage Support
* Resume Sharing via Public Links

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/shreshtabhatnagar-max/Resume_builder.git
```

### Backend Setup

```bash
cd backend

mvn clean install

mvn spring-boot:run
```

### Frontend Setup

```bash
cd frontend

npm install

npm run dev
```
